# React Portfolio Dashboard

A minimal two-page React dashboard that highlights project stats plus a portfolio gallery. Built with Vite and React Router for a lightweight starting point.

## Scripts

- `npm install` – install dependencies
- `npm run dev` – run the local dev server (default https://localhost:5173)
- `npm run build` – build the optimized production bundle
- `npm run preview` – locally preview the production build

## Structure

- `src/main.jsx` – React entry point
- `src/App.jsx` – layout + router
- `src/components` – reusable UI pieces (NavBar, StatCard)
- `src/pages` – overview dashboard and dedicated portfolio gallery
- `src/styles/global.css` – shared theming
